HTML Soup: Using every HTML tag that isn't already in use on my website

Creating Custom HTML Elements: Hello World

Using JavaScript, I defined and created a custom HTML element called <hello-world> that, when added to a page, prints "Hello World!" to the console without ever modifying the DOM. You can tell I didn't modify the DOM, because I put this element between this section's heading and text, and you can't see a space that it takes up!

Creating Custom HTML Elements Using CSS

By using a custom HTML tag, and applying styles to that tag's children, it's possible to make unique reusable components. The social media widgets below use this technique along with inline styling. This is not ideal, but I accept it for the purpose of my assignment (using every possible tag, including <style> ). The source code includes three themes: dark, pink, and light, but I chose not to implement light because:

  1. I don't have an svg for the LinkedIn logo.

  2. I don't know how to change svg colors in CSS (yet).
